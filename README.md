# Dental Clinic Management System

## Overview
A simple full-stack dental clinic management system built with Express.js, MySQL, EJS, Bootstrap, and session-based authentication.

## Features
- User authentication with bcrypt and Express Session
- Role-based access for Admin, Subadmin, and Client
- Patient management
- Appointment management
- Inventory management
- Reports page
- Settings page
- MySQL schema and seed data

## Setup
1. Create a MySQL database.
2. Import the schema from schema.sql.
3. Import sample data from sample-data.sql.
4. Copy .env.example to .env and update your database and SMTP/Twilio configuration.
5. Install dependencies with npm install.
6. Run the app with npm start.
